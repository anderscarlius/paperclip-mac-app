# First Successful Run Checklist

## Required

- Workspace selected
- Read-only flow confirmed
- Request prepared
- Limited metadata collected
- First workspace summary shown
- Safety copy visible

## Optional

- README excerpt read
- Manifest fields read
- Improved summary shown
- Feedback questions answered

## Success statement

The first private-alpha run is successful when Paperclip produces a conservative summary from safe metadata and clearly explains what it inspected and did not inspect.
