# Next Detailed Prompt Template

Use this template when continuing the lab:

1. Objective
- What exact user-environment problem should we improve next?

2. Scope
- Which screen, workflow, or runtime path is in scope?

3. Current behavior
- What happens today, and what feels wrong or slow?

4. Expected outcome
- What should feel better after the next change?

5. Constraints
- What must stay unchanged?

6. Evidence
- Include logs, screenshots, repro steps, or exact observations if available.

7. Verification
- Which test, build, or manual check should prove the improvement?
